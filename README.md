# IT Framework Learning Toolkit

An online learning toolkit for DISCS students covering modern IT management frameworks. Built as a group output for Module 3.

## What This Is

A static website that serves as a self-contained learning resource. Students can read through the framework modules, study with flashcards, test themselves with a quiz, and generate a completion certificate — all in one place, no login required.

## Features

- Learning modules with tabbed/accordion layout
- Glossary with live search
- Flashcard study mode
- Multiple choice quiz with progress tracking and score summary
- Completion certificate generated on high score
- Downloadable cheat sheet (PDF)
- Course comparison table for the two evaluated online courses
- Reading progress indicator
- Dark mode toggle
- Mobile responsive

---

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, custom properties)
- Vanilla JavaScript
- Google Fonts
- Hosted on Vercel (auto-deploys on push to `main`)

## Getting Started

No build tools or dependencies required. Just clone the repo and open `index.html` in a browser.

```bash

cd it-framework-learning-toolkit
```

To see the live version with Vercel auto-deploy, push to the `main` branch.
